SQL Database Design Assignment

📌 About

This repository contains a relational database design and implementation assignment completed in SQL. It includes multiple SQL schema and query scripts, supporting documentation, and screenshots demonstrating the database output.

The project shows how to design and populate SQL databases, structure tables with appropriate keys and constraints, and demonstrate data retrieval using queries.

📂 Repository Contents

File	Description

- SQLfile1.sql	SQL script with database definitions and operations

- SQLfile2.sql	Additional SQL commands or schema

- SQLfile3.sql	Extended SQL script (e.g., more queries or enhancements)
  
- a2-2025-moviesdb.sql	Dedicated SQL script for a movies database
  
- a2-2025-v1.pdf	Assignment document explaining design decisions and queries
  
- SQLOutputScreenshot.docx	Screenshot(s) showing query results or database output

These components demonstrate your understanding of database creation, data manipulation and working with SQL engines.

🛠 How to Use

To run and test the SQL scripts:

1. Install a SQL database engine such as PostgreSQL, MySQL, SQL Server, or SQLite.

2. Open your database client (e.g., pgAdmin, MySQL Workbench).

3. Load and execute any of the .sql files to create tables and insert data.

4. Run sample queries to inspect data structures and results.

🎯 Key Concepts Demonstrated

This project highlights:

1. Relational schema design

2. View Table build up

3. Foreign key constraints to enforce relationships

4. Database Indexing and analysation

These are core skills for structured data management and backend fundamentals.
